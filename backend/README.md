# 문의 자동화 백엔드 설정 가이드

홈페이지에서 **문의 남기기** 버튼을 누르면 아래가 자동으로 처리됩니다.

```
고객이 문의 폼 제출
   ↓
① 구글 시트에 문의 내용 자동 저장 (이름/연락처/기종/유형/내용)
   ↓
② 사장님 휴대폰으로 문자(SMS) 즉시 알림
   ↓
③ 고객에게 "접수 완료" 카카오 알림톡 또는 문자 자동 발송
```

서버 비용 **0원**(구글 무료), 문자 비용만 건당 8~25원 수준입니다.
전체 설정은 약 10~20분 걸립니다.

---

## 1단계. 구글 시트 + Apps Script 만들기 (5분)

1. [sheets.google.com](https://sheets.google.com) 에서 **새 스프레드시트** 생성 → 이름을 "폰드림 문의목록" 등으로 변경
2. 메뉴 **확장 프로그램 → Apps Script** 클릭
3. 편집기에 기본으로 있는 코드를 모두 지우고, 이 폴더의 **`apps-script.gs`** 내용을 전부 붙여넣기
4. 코드 상단 `CONFIG`에서 아래 값 수정:
   - `ADMIN_PHONE`: 알림 받을 사장님 번호 (예: `01012345678`)
   - `SENDER_PHONE`: 발신번호 (문자업체에 등록할 번호와 동일하게)
   - `ADMIN_TOKEN`: 아무 긴 문자열 (관리자페이지 조회용 비밀번호)
5. 저장(💾) 후 **배포 → 새 배포** 클릭
   - 유형: **웹 앱**
   - 실행 계정: **나**
   - 액세스 권한: **모든 사용자** ← 중요!
6. 배포하면 나오는 **웹 앱 URL**(`https://script.google.com/macros/s/.../exec`)을 복사

> ⚠️ 코드를 수정한 뒤에는 **배포 → 배포 관리 → 새 버전**으로 다시 배포해야 반영됩니다.

## 2단계. 홈페이지에 연결 (1분)

`assets/js/config.js` 파일을 열고:

```js
inquiryEndpoint: "https://script.google.com/macros/s/여기에복사한URL/exec",
```

이것만 하면 **① 시트 저장**은 바로 작동합니다.

## 3단계. 문자(SMS) 발송 연동 (10분)

문자 발송 업체는 둘 중 하나를 선택하세요. (개인사업자 가입 가능)

### 옵션 A — 알리고 (추천: 가장 저렴·간단, SMS 건당 약 8.4원)

1. [smartsms.aligo.in](https://smartsms.aligo.in) 회원가입
2. **발신번호 사전등록** (통신서비스 이용증명원 필요 — 본인 명의 번호면 간단)
3. 마이페이지에서 **API Key 발급** + 홈페이지 서버 IP 대신 **모든 IP 허용** 또는 구글 IP 대역 등록
4. `apps-script.gs`의 CONFIG 수정:
   ```js
   SMS_PROVIDER: "aligo",
   ALIGO_API_KEY: "발급받은 키",
   ALIGO_USER_ID: "알리고 아이디",
   ```

### 옵션 B — 솔라피 (카카오 알림톡까지 쓰려면 이쪽)

1. [solapi.com](https://solapi.com) 회원가입 → 발신번호 등록
2. **API Key / API Secret** 발급
3. CONFIG 수정:
   ```js
   SMS_PROVIDER: "solapi",
   SOLAPI_API_KEY: "...",
   SOLAPI_API_SECRET: "...",
   ```

설정 후 Apps Script 편집기에서 `testSms` 함수를 실행하면 사장님 번호로 테스트 문자가 갑니다.

## 4단계. (선택) 고객에게 카카오 알림톡 보내기

문자 대신 카카오톡으로 "접수 완료" 메시지를 보내려면:

1. [카카오톡 채널](https://center-pf.kakao.com) 개설 (매장 채널 — 상담용으로도 필수)
2. 솔라피 콘솔에서 **카카오 채널 연동** → `pfId` 발급
3. **알림톡 템플릿 등록** 후 카카오 승인(1~3일) 대기
   - 템플릿 예시: `[폰드림] #{이름}님, 문의가 정상 접수되었습니다. 영업시간 내 #{상담방법}로 빠르게 연락드리겠습니다.`
4. CONFIG 수정:
   ```js
   KAKAO_PF_ID: "발급받은 pfId",
   KAKAO_TEMPLATE_ID: "승인된 템플릿 ID",
   ```

> 알림톡 발송이 실패하면 자동으로 문자(SMS)로 대체 발송되도록 코드에 처리되어 있습니다.
> 알림톡 미설정 상태에서도 `SEND_CUSTOMER_REPLY: true`면 고객에게 문자로 발송됩니다.

## 5단계. 관리자 페이지에서 문의 확인

`admin.html`을 열면 접수된 문의 목록을 볼 수 있습니다.

1. 첫 접속 시 **Apps Script URL**과 **ADMIN_TOKEN**(1단계에서 정한 값) 입력
2. 이후 자동 저장되어 바로 목록이 보입니다.

구글 시트에서 직접 봐도 되고, 시트의 "처리상태" 열에 완료/진행중 등을 적어 관리하세요.

## 광고 전환 추적 (메타 · 유튜브)

`assets/js/config.js`에 ID만 넣으면 전 페이지에 자동 설치됩니다.

| 항목 | 발급처 | 입력할 곳 |
|---|---|---|
| 메타 픽셀 | [Meta 이벤트 관리자](https://business.facebook.com/events_manager) | `metaPixelId` |
| GA4 (유튜브/구글 광고) | [analytics.google.com](https://analytics.google.com) | `ga4Id` (G-XXXX) |

문의 폼 제출이 완료되면 자동으로 **메타 `Lead` 이벤트**와 **GA4 `generate_lead` 이벤트**가 발생합니다.
광고 관리자에서 이 이벤트를 전환 목표로 잡으면 "문의 남긴 사람" 기준으로 광고가 최적화됩니다.

## 자주 발생하는 문제

| 증상 | 원인/해결 |
|---|---|
| 폼 제출 시 "전송 실패" | 배포 액세스 권한이 "모든 사용자"인지 확인. 코드 수정 후 새 버전 재배포 했는지 확인 |
| 시트엔 저장되는데 문자가 안 옴 | 발신번호 사전등록 여부, API 키, 충전 잔액 확인. Apps Script 편집기 → 실행 로그 확인 |
| 알림톡이 문자로만 감 | 템플릿이 아직 카카오 승인 전이거나 templateId 오타 |
| admin.html에서 unauthorized | ADMIN_TOKEN이 apps-script.gs의 값과 다름 |
